## Development Principles of this app

1. Code shall be easy to maintain
2. Code shall be extensible for new features and functions
3. Code shall be readable by human or AI
4. Obscure business rules will be described with comments
5. Unit tests shall be provided app code with each push
6. CI/CD shall be a day one capability
7. Test automation shall be a day one capability
8. App security shall be a day one capability
